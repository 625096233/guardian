Guardian
=====

**让你的程序快速接入Zookeeper来实现高可用**

# 介绍

- 使用curator 连接zookeeper，保证连接质量。
- 实时更新zookeeper的节点数据，保证消费和生产一致。
- 实现了服务治理，目前提供了 随机 和 轮询两种方式，后续会加入 `权重` 和 `熔断`两种消费方式。
- sample 是一个示例程序，欢迎大家下载使用。
